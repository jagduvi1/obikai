# Getting help

Thanks for using Obikai! Here's where to go:

- **Questions / ideas / "how do I…"** → [GitHub Discussions](https://github.com/jagduvi1/obikai/discussions).
- **Bugs** → open an [issue](https://github.com/jagduvi1/obikai/issues/new/choose) using the Bug report template.
- **Feature requests** → open an issue using the Feature request template.
- **Self-hosting** → see the README quick-start and `docs/` (self-host guide forthcoming).
- **Security vulnerabilities** → **do not** open a public issue; follow [SECURITY.md](SECURITY.md).
- **Contributing** → [CONTRIBUTING.md](CONTRIBUTING.md). Note the contribution agreement (CLA vs no-CLA) is still being decided, so external PRs can't be merged yet — but discussion and draft PRs are welcome.

Please be kind and follow the [Code of Conduct](CODE_OF_CONDUCT.md).
